# Individual Household Electric Power Consumption Dataset

## Source
- Repository: UCI Machine Learning Repository
- URL: https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption
- DOI: https://doi.org/10.24432/C58K54

## Description
This dataset contains household electric power consumption measurements recorded
at one-minute intervals over several years. It is commonly used for time-series
and data stream analysis.

## Usage in This Research
The dataset is used for evaluating streaming forecasting and drift detection
algorithms under non-stationary conditions.
