Source

DOI: https://doi.org/10.24432/C58K54

Description

This dataset contains household electric power consumption measurements recorded at one-minute intervals over multiple years. It is widely used for time-series forecasting, data stream analysis, and non-stationary learning studies.

Usage in This Research

The dataset is used to evaluate streaming forecasting, adaptive learning, and drift detection algorithms under non-stationary conditions.

Download Instructions

The dataset can be downloaded directly from the official UCI Machine Learning Repository using the link provided above.
